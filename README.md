# File_Explorer
# Android 文件管理器

#发布版本
####[腾讯助手](http://android.myapp.com/myapp/detail.htm?apkName=com.yang.file_explorer)      

#包常用ui控件有:        
1.侧滑菜单slidingMenu           
2.兼容actionbar----actionbarsherlock        

#项目结构：        
com.yang.file_explorer     
                ---------------adapter    适配器        
                ---------------apis       文件交互操作api        
                ---------------entity     实体类              
                ---------------interfaces 接口类             
                ---------------ui         Activity类       
                ---------------view       Fragment类        
                ---------------widget     自定义控件      
                ---------------utils      工具类       

#目前app文件功能：         
1.新建文件         
2.删除文件       
3.刷新文件     
4.文件排序     
5.文件分享    
6.文件的复制、剪切        
7.文件重命名     
8.文件搜索                 
9.收藏文件并浏览                 
10.ftp文件上传
